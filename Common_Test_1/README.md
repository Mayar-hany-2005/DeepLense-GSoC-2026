# Common_Test_I Documentation

This folder contains the multi-class lens classification pipeline and its associated artifacts for detecting dark matter substructures (`no_sub`, `subhalo`, `vortex`).

## Files

* **`Common_Test_I_Multi_Class_Classification.ipynb`**
  * Jupyter notebook implementing the multi-class classification pipeline.
  * Explains the data preprocessing (1-channel to 3-channel conversion), model architectures (ResNet18 & EfficientNet-B0), and evaluation steps.
  * Produces multi-class Macro-Average ROC curves, AUC scores, and model comparison plots as outputs.
  * **To use:** Open in Jupyter/Colab, mount Google Drive (if needed for the dataset), and run all cells sequentially.

* **`best_model_efficientnet_b0.pth` & `best_model_resnet18.pth`**
  * Trained PyTorch model weights for both classification pipelines.
  * Generated by the training process described in the notebook.
  * *(Note: Due to GitHub file size limits, these might be hosted via an external Google Drive link provided in the notebook/main repo).*
  * **To use:** The notebook loads these weights automatically for the final evaluation and ROC curve generation.

## Outputs
* One-vs-Rest (OvR) Macro-Average ROC plots, training history curves, and model comparison charts are generated by running the notebook.
* Outputs are displayed inline and automatically saved as `.png` images during execution.

## Reviewer Notes
* Ensure the dataset (`dataset.zip`) is available and the extraction path in the first cell matches your environment.
* All steps for reproduction, from data loading to final evaluation, are automated within the notebook's cells.
